## Component Details

### 1. 3D Tilt & Parallax Cards
* **Purpose**: Create tactile feedback and visual depth on categories and hero assets without adding video weight.
* **Desktop Appearance**: Cards rotate in 3D perspective based on pointer position. A subtle reflective gradient overlay moves opposite to the cursor, creating a "glass glare" sheen.
* **Mobile Appearance**: Rotations are disabled on touch devices to prevent accidental triggers. Cards fall back to smooth standard tap scales (`scale(0.98)`).
* **Animation Details**: Uses CSS `transform: perspective(1000px) rotateX() rotateY()` updated smoothly on mousemove, transitioning back to baseline (`0deg`) with a soft spring ease-out (`0.5s cubic-bezier(0.16, 1, 0.3, 1)`) on mouseleave.
* **Performance Impact**: Extremely light. Operates purely on layout-neutral CSS properties (`transform`). 0% reflow cost.

### 2. Magnetic CTA Buttons
* **Purpose**: Draw user attention to key actions (Browse Products, Contact Us, Checkout).
* **Desktop Appearance**: Buttons gently glide towards the pointer when it enters a 40px radius.
* **Mobile Appearance**: Disabled. Buttons remain fixed.
* **Animation Details**: Tracks cursor distance from button center and translates button coordinates by up to 10px. Relies on smooth transitions (`transition: transform 0.2s ease-out`).
* **Performance Impact**: Negligible. JS only updates CSS transform offsets during hover states.

### 3. Product Carousel
* **Purpose**: Enable users to browse products naturally without cluttering the homepage vertical space.
* **Desktop/Mobile Appearance**: Touch/trackpad scrollable row of product cards with a masked container. Subtle gradient masks fade the left and right edges.
* **Animation Details**: Scroll snap transitions anchor cards to center alignment. Arrow buttons slide coordinates by card-width values.
* **Performance Impact**: Zero JS CPU cost for scrolling; leverages GPU-accelerated CSS Scroll Snap (`scroll-snap-type: x mandatory`).

### 4. Quick View Modal
* **Purpose**: Streamline purchase flow, allowing product detail review and cart addition without leaving the homepage.
* **Desktop/Mobile Appearance**: A centered modal overlay with a blurred background overlay (`backdrop-blur`).
* **Animation Details**: Modal scales up (`scale(0.95) -> scale(1)`) and fades in over `0.3s cubic-bezier(0.34, 1.56, 0.64, 1)`. Backdrop blurs progressively.
* **Performance Impact**: Smooth 60 FPS transitions since opacity and scale transforms bypass layout/paint cycles.
